# CLI Usage

## Calculator

```powershell
open-ai-econ calculator
```

## Run a scenario

```powershell
open-ai-econ run scenarios/basic_usage.yaml --plot

```

## Compare Models

```powershell
open-ai-econ compare --models gpt-5.4 gpt-5.4-mini --input-tokens 1200 --output-tokens 400 --calls-per-day 50000
```
